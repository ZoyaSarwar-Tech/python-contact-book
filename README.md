# 📒 Contact Book App in Python

A simple command-line Contact Book application built with Python. It allows users to store and manage contacts using a dictionary.

This project is perfect for Python beginners who want to practice dictionaries, functions, loops, and conditional statements.

---

## 🚀 Features

- ➕ Add a Contact
- 📋 View All Contacts
- 🔍 Search a Contact
- 🗑️ Delete a Contact
- ❌ Exit the Program
- 📱 Simple Menu-Driven Interface

---

## 🛠️ Technologies Used

- Python 3

No external libraries are required.

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/ZoyaSarwar-Tech/python-contact-book.git
```

2. Open the project folder

```bash
cd python-contact-book
```

3. Run the program

```bash
python contact_book.py
```
